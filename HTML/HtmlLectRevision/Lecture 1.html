To Make Text Bold use <b> tag
To make text strong (same as bold) use <strong> tag
To Make Text Italic use <i> tag
To make text Emphasized (same as Italic) use <em> tag
To Make Text Underlined use <u> tag
To highlight the text use <mark> tag
To represent deleted text use <del> tag
To represent inserted text use <ins> tag
