CSS Font Properties

  1. Font-Family
      We can use different type Web Fonts in our HTML document using font-family in CSS.
    • Verdana
    • Arial
    • "Arial Black" Etc...
  2. Font-Size
      We can use different types of units to define the font size im our HTML document using font-size in CSS.
    • 1em = 12pt = 16px = 100% (measurement)
    • 2em (that means 2 times the size of the current font)
    • 20pt (Points[1pt = 1/72 of 1 inch])
    • 5px (Pixels[1px = 1/96 of 1 inch])
    • 200% (Percentage[it is used when we have to relate font size of a tag to a perent tag])
  3. Font-Style
      We can give different types of Font styles in our HTML document using Font-Style in CSS.
    • Italic
    • Oblique
    • Normal Etc...
  4. Font-Weight
      We can give different types of Font Weight in our HTML document using Font-Weight in CSS.
    • Bold
    • Normal Etc...
